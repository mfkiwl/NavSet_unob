Modulators Package¶

modulators.constallation_mappers module¶

Functions of the module Constallation Mapper produce baseband signals according to required passband modulations and shape of the pulse.

At this moment available choices are BPSK or QPSK modulations and three different pulses - rectangular, sinus cardinale or raised cosine.

Returned baseband signals are numpy arrays of the same size as the time axis t.

Parameters:

Key word Possible values Default Description
